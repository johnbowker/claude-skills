# Listing Opportunities Using Data

Using provided internal sources (insights, NPS verbatims, support logs, usage drop-offs), identify a list of specific user opportunities (problems or unmet needs) that are blocking progress toward a stated outcome.

## Instructions

Cluster findings into 3-5 themes with supporting quotes or evidence.

### Context to Gather
- User research data or source material
- The refined outcome statement

### Output Format
A set of 3-5 opportunity clusters with user-backed evidence.

### Considerations
- Prioritize real-world user pain
- Highlight issues the organisation has the ability to address today

Ask the user for their data and outcome statement if not provided.
