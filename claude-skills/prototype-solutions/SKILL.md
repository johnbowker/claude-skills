# Prototyping Potential Solutions

Take high-priority solutions and sketch the end-to-end user journey for each.

## Instructions

For each solution provided:
1. Include key steps, touchpoints, and where value is delivered
2. Focus on the specified personas
3. Highlight potential moments of friction or delight

### Context to Gather
- The solution concepts to prototype
- Persona or use case
- Assumptions being tested

### Output Format
Annotated flows or journey descriptions.

### Considerations
- Favor speed and clarity
- Use this to identify moments to test and learn from

Ask the user for their solutions and persona details if not provided.
