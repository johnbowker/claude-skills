# Generate User Personas

Act as a senior UX researcher. Based on provided data about a target audience, generate 3 detailed user personas.

## Instructions

For each persona, include:
1. **Name and brief bio** — A realistic character sketch
2. **Goals** — What they're trying to achieve
3. **Frustrations** — What blocks or annoys them
4. **Motivations** — What drives their behavior
5. **Context** — Their environment, tools, and constraints

### Context to Gather
- Product type or description
- User data, survey results, or market research

### Output Format
3 detailed personas with structured sections for goals, frustrations, motivations, and bio.

Ask the user for their product type and any available user data if not provided.
