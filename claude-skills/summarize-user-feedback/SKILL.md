# Summarize User Feedback

Analyze provided user feedback and extract actionable insights.

## Instructions

1. Identify the **top 5 recurring themes**
2. For each theme, determine **sentiment** (positive, negative, neutral)
3. Provide **illustrative quotes** from the raw data
4. Suggest **priority actions** based on frequency and severity

### Context to Gather
- Source of feedback (App Store reviews, survey responses, support tickets, etc.)
- Raw feedback data

### Output Format
Structured analysis with themes, sentiment indicators, supporting quotes, and recommended actions.

### Considerations
- Look for patterns, not just individual complaints
- Note any contradictions between user segments
- Highlight quick wins vs. systemic issues

Ask the user for their feedback source and data if not provided.
