# Outlining / Brainstorming Potential Solutions

Based on a prioritized opportunity, generate a diverse set of potential solutions.

## Instructions

Think beyond just UI changes and include:
- Changes to user flow
- Feature adjustments
- Education, onboarding, or support mechanisms

Return 5+ creative and practical options, with 1-2 sentence descriptions of each.

### Context to Gather
- Opportunity statement
- Any past solutions that have worked or failed
- Product constraints or boundaries (if any)

### Output Format
Broad range of possible solution concepts.

### Considerations
- Quantity over quality at this stage
- Use analogy from other companies if helpful

Ask the user for their opportunity statement if not provided.
