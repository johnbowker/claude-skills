# Create Interview Questions

Generate 10 open-ended interview questions to uncover deep insights about user pain points.

## Instructions

Questions should:
1. Be open-ended (not yes/no)
2. Avoid leading the participant
3. Progress from broad context to specific pain points
4. Include follow-up probes where appropriate
5. Cover behavior, emotions, and workarounds

### Context to Gather
- Product name or description
- Specific problem area to explore

### Output Format
10 numbered questions with optional follow-up probes.

### Tips
- Start with warm-up questions about their role and context
- Move to specific workflows and pain points
- End with aspirational questions about ideal solutions

Ask the user for their product and problem area if not provided.
