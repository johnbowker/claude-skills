# Draft a Product Vision

Help draft a compelling product vision statement for a product.

## Instructions

The vision should be:
1. **Inspiring** — Motivates the team and stakeholders
2. **Concise** — One to two sentences maximum
3. **Future-focused** — Describes the long-term impact, not current features
4. **Customer-centred** — Frames value from the user's perspective

### Process
1. Ask about the product's core function and target audience
2. Explore the long-term impact they want to achieve
3. Draft 3 vision statement options
4. Refine based on feedback

### Output Format
3 vision statement options with brief rationale for each.

### Examples of Good Vision Statements
- "A world where every person can freely share knowledge" (Wikipedia)
- "To give everyone the power to create anything" (Canva)

Ask the user for their product description and target audience if not provided.
