# Test My Hypothesis

Use the provided hypothesis and create a scenario that can be used to test if the direction is correct.

## Instructions

Given:
- **My Hypothesis**: The user's stated hypothesis
- **My Target Customer**: The user's target customer description

Create a realistic, customer-centred scenario that:
1. Simulates the target customer's environment and context
2. Tests whether the hypothesis holds under real-world conditions
3. Identifies potential failure points or edge cases
4. Suggests observable signals that would confirm or deny the hypothesis

Ask the user for their hypothesis and target customer if not provided.
