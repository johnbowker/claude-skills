# Create a PRFAQ Document

Create a PRFAQ document from a provided hypothesis.

## Instructions

1. Write a compelling **Press Release** section that announces the product/feature as if it's already launched
2. Populate the **FAQ section** with questions from the specified personas (typically 2)
3. Each persona should have 3-5 questions that reflect their unique concerns and perspective

### Structure

**Press Release:**
- Headline
- Subheadline
- Problem statement
- Solution description
- Customer quote
- Call to action

**FAQ:**
- Questions grouped by persona
- Each answer should be concise and address the persona's specific concern

### Context to Gather
- The hypothesis or product idea
- Persona 1 description
- Persona 2 description

Ask the user for their hypothesis and persona descriptions if not provided.
