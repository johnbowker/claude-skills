# Define North Star Metric

Help brainstorm potential North Star Metrics that align with delivering core user value.

## Instructions

For each suggested metric:
1. **Name the metric** clearly
2. **Explain why** it's a good candidate
3. **Describe how** it connects to user value
4. **Note limitations** or risks of over-optimizing for it

### Criteria for a Good North Star Metric
- Reflects the core value delivered to customers
- Is a leading indicator of revenue
- Can be influenced by the product team
- Is measurable and trackable
- Captures the "aha moment" or ongoing value

### Context to Gather
- Product description
- Business model (subscription, marketplace, etc.)
- Current growth stage

### Output Format
3-5 candidate metrics with rationale, ranked by recommendation strength.

Ask the user for their product description if not provided.
