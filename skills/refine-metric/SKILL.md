# Refine My Metric

Use the provided metric and help refine it to ensure it focuses on the outcome, is well-suited to the customer, and is something worth tracking.

## Instructions

Given:
- **My Metric**: The user's current metric
- **My Target Customer**: The user's target customer description

Evaluate the metric against these criteria:
1. Does it measure an outcome (change in customer behavior) rather than an output (feature shipped)?
2. Is it aligned with the target customer's success?
3. Is it actionable — can the team influence it?
4. Is it measurable with available data?

Offer at least 3 alternative metrics if the current one doesn't align well. For each alternative, explain why it better captures the intended outcome.

Ask the user for their metric and target customer if not provided.
