# SWOT Analysis

Act as a business strategist. Conduct a SWOT analysis for the provided product.

## Instructions

Be critical and realistic. For each quadrant provide 3-5 points:

1. **Strengths** — Internal advantages and differentiators
2. **Weaknesses** — Internal limitations and gaps
3. **Opportunities** — External factors that could be leveraged
4. **Threats** — External risks and competitive pressures

### Additional Analysis
- Identify the most critical item in each quadrant
- Suggest 2-3 strategic actions based on the analysis
- Note any strengths that could address threats (or weaknesses that amplify them)

### Context to Gather
- Product name
- Brief product description
- Target market and competitors (if known)

### Output Format
Structured SWOT matrix with strategic recommendations.

Ask the user for their product name and description if not provided.
